## Context
System prompts split across multiple blocks or delivered as formatted
markdown require manual reassembly and introduce copy errors.

## Options Considered
Options not explicitly recorded in original decision entry.

## Decision
All system prompts generated by the Architect are delivered as a single
copyable code block.

## Rationale
Not explicitly recorded in original entry.

## Consequences
Rules out delivering prompts as formatted markdown. Rules out splitting
a prompt across multiple message blocks. One code block, copy, paste, done.
